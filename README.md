# 🕉️ Mahadev Ashirbad – Spiritual Website

> **"Mahadev Ashirbad – Where Faith Inspires Every Heart."**

A modern, responsive, and interactive website dedicated to **Lord Shiva (Mahadev)**. This project is built using **HTML5, CSS3, and JavaScript** with a focus on beautiful UI, smooth animations, and a peaceful spiritual experience.

---

## 📖 Overview

**Mahadev Ashirbad** is a front-end web project that combines spirituality with modern web design. It includes animated visuals, a live digital clock, inspirational Shiva quotes, interactive flower offerings, and a clean responsive layout suitable for desktop and mobile devices.

---

## ✨ Features

* 🕉️ Beautiful Mahadev-themed landing page
* 🌙 Modern dark spiritual UI
* 💬 Random Mahadev motivational quotes
* 🌺 Interactive flower offering animation
* 🪔 Animated diya
* 📱 Fully responsive design
* 🎨 Smooth CSS animations
* ⚡ Lightweight and fast performance
* 🌐 Pure HTML, CSS, and JavaScript (No frameworks)
